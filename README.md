# blog
# 🎮 My Game Development & Review Blog

HTML, CSS, JavaScript만을 사용하여 프론트엔드를 직접 코딩하고 디자인한 **나만의 개인 게임 블로그** 프로젝트입니다. 게임 개발 일지(Python, Ursina, Roblox 스튜디오 등)와 게임 공략, 리뷰를 깔끔하게 기록하기 위해 제작되었습니다.

---

## 🚀 주요 기능 (Features)

*   **시맨틱 마크업(Semantic Markup):** `<header>`, `<nav>`, `<main>`, `<article>`, `<footer>` 등의 태그를 사용하여 검색 엔진과 가독성에 최적화된 웹 표준 구조 설계
*   **다크 모드 테마(Dark Theme):** 장시간 글을 읽거나 코드를 봐도 눈이 편안한 게이머 감성의 어두운 UI 테마
*   **개발자용 코드 블록(Code Block):** `<pre>`와 `<code>` 태그를 활용하여 개발 소스 코드를 가독성 있게 표현
*   **하단 미니 한마디 상자(Mini Notice Box):** 블로그 메인 하단(Footer 위)에 최근 개발 현황이나 공지사항을 5줄 내외로 간결하게 보여주는 고정 컴포넌트
*   **반응형 웹 디자인(Responsive Design):** 데스크톱 PC뿐만 아니라 태블릿(Galaxy Tab, iPad) 등 다양한 디바이스 환경을 고려한 레이아웃
